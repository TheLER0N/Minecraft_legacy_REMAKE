В PowerShell это так.

**Через полный путь к MSBuild**
```powershell
cd "C:\Users\Пользователь\Desktop\Minecraft legacy\Minecraft legacy"

# Debug
& "C:\Program Files\Microsoft Visual Studio\18\Insiders\MSBuild\Current\Bin\MSBuild.exe" "Minecraft legacy.vcxproj" /p:Configuration=Debug /p:Platform=x64

# Release
& "C:\Program Files\Microsoft Visual Studio\18\Insiders\MSBuild\Current\Bin\MSBuild.exe" "Minecraft legacy.vcxproj" /p:Configuration=Release /p:Platform=x64
```

**Если открыт exe и сборка падает с `LNK1168`**
```powershell
taskkill /F /IM "Minecraft legacy.exe"
```

**Если ты запускаешь именно `Developer PowerShell for Visual Studio`**
Можно короче, без полного пути:
```powershell
cd "C:\Users\Пользователь\Desktop\Minecraft legacy\Minecraft legacy"

msbuild "Minecraft legacy.vcxproj" /p:Configuration=Debug /p:Platform=x64
msbuild "Minecraft legacy.vcxproj" /p:Configuration=Release /p:Platform=x64
```

Готовые файлы будут тут:
- Debug: `x64\Debug\Minecraft legacy.exe`
- Release: `x64\Release\Minecraft legacy.exe`

Если хочешь, могу ещё написать тебе одну короткую команду `.ps1`, которая собирает и Debug, и Release сразу.
